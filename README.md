# prerequisite
```
# jdk8 버전이 필요합니다.
# JAVA_HOME 및 PATH 환경변수가 설정되어있어야합니다.

$ sudo apt update
$ sudo apt install openjdk-8-jdk

$ java -version
```

# Build
```
$ git clone https://github.com/kys6879/point-api.git
$ cd point-api

$ ./build.sh
```

# Run
```
$ cd point-api
$ java -jar application.jar
```
